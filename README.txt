KeySMS for WordPress.

== Description ==

KeySMS is a plugin to send SMS messages to your customers. The plugin is free but you need an account at KeySMS to use the service.

[Sign up for a free demo account](https://app.keysms.no/demo).

== Installation ==

1. Activate the plugin.
2. Navigate to Settings -> KeySMS
3. Enter your username and API key
